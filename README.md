# 🤖 AI Virtual Assistant

An intelligent AI-powered virtual assistant that allows users to interact using voice commands, customize their own AI assistant, and receive AI-generated responses powered by Google's Gemini AI.

## 📌 Features

### 🔐 Authentication System
- User Registration
- User Login
- JWT Authentication
- Secure HTTP-Only Cookies
- Protected Routes

### 🎙️ Voice Assistant
- Voice Command Recognition
- Speech-to-Text Integration
- AI-powered Responses
- Real-time Interaction

### 🤖 AI Customization
- Select Assistant Avatar
- Upload Custom Assistant Images
- Set Personalized Assistant Names
- Save Preferences in Database

### ☁️ Cloud Integration
- Cloudinary Image Storage
- Secure Image Uploads
- User Profile Management

### 🧠 Gemini AI Integration
- Natural Language Understanding
- Conversational Responses
- Intelligent Query Handling
- Context-aware Assistance

### 📱 Responsive Design
- Mobile Friendly
- Tablet Friendly
- Desktop Optimized
- Modern Glassmorphism UI
